# Gyeonggi Metropolitan Bus as Seen by City Big Data

## Demand Pattern Analysis and Operational Efficiency Strategy

This project analyzes metropolitan bus demand around the Jamsil Metro Area Bus Transfer Center using urban big data. It identifies demand patterns and proposes operational efficiency strategies through bus supply reallocation.

**Authors:** Keum Kyoung Hoon, Young-Long Kim  
**Affiliation:** Department of Smart City, Gachon University  

## Files
- `gyeonggi_metropolitan_bus_demand_analysis.ipynb`: Data preprocessing, analysis, visualization, and optimization workflow
- `gyeonggi_metropolitan_bus_demand_analysis.pdf`: Conference paper

## Methods
- Transportation card data analysis
- Real-time bus arrival data analysis
- Multiple regression analysis
- Integer programming–based bus supply reallocation
